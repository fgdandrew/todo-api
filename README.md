# Простое API для управления задачами
Реализация REST API для управления списком задач (To-Do List) на PHP с использованием Laravel включает:
- CRUD-операции для задач (Создание задачи, просмотр списка задач, просмотр одной задачи, обновление задачи, удаление задачи).
- Валидация данных.
- SQLite в качестве базы данных.
